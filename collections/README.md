# 大型汇编材料库

用途：归档多校、多套、跨主题或页数较多的 Markdown/PDF 汇编成品。

本目录用于保存不适合放入单份试卷 `exams/`、也不适合放入小专题讲义 `handouts/` 的大型资料集合。

归档约定：

- 先按资料类型建立中文分类目录，例如 `多校试卷解析汇编/`、`综合题集汇编/`。
- 每个汇编成品单独一个中文文件夹，不把不同来源、不同用途或明显不是一类的资料混在一起。
- 文件夹内至少包含一份语义化命名的 `.md` 和对应 `.pdf`。
- 若 Markdown 引用图片，必要图片放在同级 `figures/` 目录内。
- 原始 OCR 目录、`.tex`、`.docx`、`result.json` 和调试中间文件不放入本目录。
