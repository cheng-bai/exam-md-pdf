# exams

这里管理试卷 Markdown、题图和 PDF。

## 最终云端归档

不带 `-工作稿` 的中文试卷目录是推送到 GitHub 管理的最终归档目录。每份试卷一个文件夹，目录名只描述试卷本身，版本类型写在文件名里：

```text
2026上海市建平中学高一期末数学试卷/
├── 2026上海市建平中学高一期末数学试卷-学生版.md
└── 2026上海市建平中学高一期末数学试卷-学生版.pdf
```

若 Markdown 引用题图，必须同时保留必要的 `figures/`。

## 审校工作稿

带 `-工作稿` 的目录，如 `2026上海市建平中学高一期末数学试卷-工作稿/`，用于保留 OCR 审校现场、质量报告、渲染输出和题图。它们不等同于最终云端归档，不要直接整体提交或公开。
