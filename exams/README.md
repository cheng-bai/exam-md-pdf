# exams

这里管理试卷 Markdown、题图和 PDF。

## 最终云端归档

`高中_*` 目录是推送到 GitHub 管理的最终归档目录。每份试卷一个文件夹，目录名和主文件名保持一致：

```text
高中_学校标准名_年级_学科_考试类型_版本类型_提交日期/
├── 高中_学校标准名_年级_学科_考试类型_版本类型_提交日期.md
└── 高中_学校标准名_年级_学科_考试类型_版本类型_提交日期.pdf
```

若 Markdown 引用题图，必须同时保留必要的 `figures/`。

## 旧工作稿

中文试卷名目录，如 `敬业中学高一期末数学试卷/`，用于保留 OCR 审校现场、质量报告、渲染输出和题图。它们不等同于最终云端归档，不要直接整体提交或公开。
